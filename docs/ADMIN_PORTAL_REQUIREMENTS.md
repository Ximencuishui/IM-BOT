# Admin Portal 功能需求说明

## 📋 概述

Admin Portal 是系统的管理员后台，提供系统级的管理功能，用于管理用户、授权码、定价、规则库、推广等核心业务。

**访问路径**: `http://your-domain.com/admin-portal`  
**部署位置**: 云服务器或公网可访问的服务器  
**目标用户**: 系统管理员、运营人员

---

## 🎯 核心功能模块

### 1. 👥 用户管理

#### 功能描述
管理系统中的所有用户账户，包括创建、编辑、删除、权限设置等。

#### 详细功能
- **用户列表**
  - 查看所有注册用户
  - 搜索和筛选（按用户名、邮箱、角色、状态）
  - 分页显示
  
- **用户操作**
  - 创建新用户（手动创建或导入）
  - 编辑用户信息（用户名、邮箱、手机号、公司名）
  - 设置用户角色（超级管理员、管理员、普通用户）
  - 禁用/启用账户
  - 删除用户（软删除，保留数据）
  
- **用户详情**
  - 查看用户基本信息
  - 查看用户活跃度统计
  - 查看登录历史
  - 查看关联的授权码
  - 查看订单历史
  
- **批量操作**
  - 批量导入用户（CSV/Excel）
  - 批量导出用户数据
  - 批量禁用/启用

#### 数据字段
```
- id: 用户ID
- username: 用户名
- email: 邮箱
- phone: 手机号
- company_name: 公司名称
- role: 角色（super_admin）
- is_active: 是否激活
- subscription_type: 订阅类型（monthly/yearly）
- max_groups: 最大群组数
- created_at: 注册时间
- last_login: 最后登录时间
```

### 2. 🔑 授权管理

#### 功能描述
管理系统中的用户授权，包括用户订阅管理、到期预警、续费处理等。

#### 详细功能
- **用户授权列表**
  - 查看所有用户的授权状态
  - 搜索和筛选（按用户、状态、到期时间、套餐类型）
  - 分页显示
  
- **授权操作**
  - 查看用户授权详情（到期时间、套餐类型、群组数量）
  - 手动延长授权有效期
  - 暂停/恢复用户授权
  - 调整套餐类型
  
- **到期预警**
  - 即将到期用户列表（7天、15天、30天）
  - 已过期用户列表
  - 到期趋势图表
  
- **授权统计**
  - 总授权用户数
  - 活跃授权用户数
  - 已过期用户数
  - 套餐销售趋势图
  - 续费率统计
  
- **用户同步**
  - 查看用户登录记录
  - 查看桌面端同步状态

#### 数据字段
```
- id: 用户ID
- email: 用户邮箱
- username: 用户名
- subscription_type: 订阅类型（monthly/yearly）
- subscription_expires_at: 订阅到期时间
- max_groups: 最大群组数
- is_active: 是否激活
- created_at: 创建时间
- last_login_at: 最后登录时间
```

### 3. 💰 定价配置

#### 功能描述
管理产品的定价策略，以组合套餐形式定义产品。

#### 详细功能

  - 编辑产品信息（名称、描述、包含内容）

  - 启用/禁用产品

- **套餐配置**
  - 定义套餐内容（功能模块、使用限制）
  - 设置折扣规则（早鸟价、团购价）
  - 配置试用期
  - 价格历史记录
  
- **定价策略**
  - 基础定价设置
  - 阶梯定价（按用户数、按用量）
  - 促销活动管理
  - 优惠券管理
  
- **价格展示**
  - 前台价格页面配置
  - 价格对比表
  - 推荐套餐标记

#### 数据字段
```
- id: 产品ID
- product_name: 产品名称
- description: 产品描述
- features: 功能特性（JSON）
- monthly_price: 月付价格
- yearly_price: 年付价格
- trial_days: 试用天数
- is_active: 是否启用
- sort_order: 排序
- created_at: 创建时间
```

### 4. 📚 规则库管理

#### 功能描述
管理系统的总规则库，包括解析规则、统计规则的增删改查，以及查看各用户的规则备份。

#### 详细功能
- **总规则库管理**
  - 创建新规则模板
  - 编辑规则（解析规则、统计规则）
  - 删除规则模板
  - 规则分类管理（按行业、场景）
  - 规则版本控制
  
- **规则模板分发**
  - 将规则模板推送给用户
  - 查看规则使用情况统计
  - 规则更新通知
  
- **用户规则备份查看**
  - 查看各用户备份的规则
  - 规则对比（用户规则 vs 总库规则）
  - 规则同步建议
  - 下载用户规则备份
  
- **用户授权监控**
  - 查看用户授权到期状态
  - 规则使用权限控制
  - 到期提醒设置
  
- **规则导入导出**
  - 批量导入规则（CSV/TXT/MD）
  - 导出规则库
  - 规则备份与恢复

#### 数据字段
```
RuleTemplate:
- id: 模板ID
- template_name: 模板名称
- industry: 所属行业
- description: 描述
- parse_rules: 解析规则（JSON）
- stat_rules: 统计规则（JSON）
- version: 版本号
- is_preset: 是否预设
- created_at: 创建时间
- updated_at: 更新时间

UserRuleBackup:
- id: 备份ID
- user_id: 用户ID
- rule_content: 规则内容
- backup_time: 备份时间
- license_status: 授权状态
```

## 🔐 权限控制

### 角色定义
- **超级管理员（super_admin）**: 所有权限
- **管理员（admin）**: 大部分管理权限，部分敏感操作受限
- **运营人员（operator）**: 用户管理、订单管理、推广管理
- **客服（support）**: 只读权限，查看用户和订单信息

### 权限矩阵
| 功能模块 | super_admin | admin | operator | support |
|---------|-------------|-------|----------|---------|
| 用户管理 | ✅ | ✅ | ✅ | 👁️ |
| 授权管理 | ✅ | ✅ | ✅ | 👁️ |
| 定价配置 | ✅ | ✅ | ❌ | ❌ |
| 规则库管理 | ✅ | ✅ | ✅ | 👁️ |
| 订单与续费 | ✅ | ✅ | ✅ | 👁️ |
| 推广管理 | ✅ | ✅ | ✅ | ❌ |
| 系统监控 | ✅ | ✅ | 👁️ | ❌ |
| 审计日志 | ✅ | ✅ | 👁️ | ❌ |
| 系统设置 | ✅ | ⚠️ | ❌ | ❌ |
| 插件管理 | ✅ | ✅ | ❌ | ❌ |

注：✅ 完全权限，⚠️ 部分权限，👁️ 只读，❌ 无权限

### 10. 🧩 插件管理

#### 功能描述
管理插件商店中的所有插件，包括插件的创建、编辑、版本管理、状态控制、安装统计等。该模块仅对超级管理员和管理员开放。

#### 详细功能
- **插件列表**
  - 查看所有插件（含已上架、未上架、已废弃）
  - 搜索和筛选（按名称、分类、行业、状态）
  - 分页显示，支持自定义每页条数
  - 列表信息：插件名称、图标、分类、行业、版本、评分、下载量、状态

- **插件创建/编辑**
  - **基础信息**
    - 插件名称（必填）
    - 插件代码（唯一标识，创建后不可修改）
    - 插件描述（必填）
    - 简短描述（列表展示用）
    - 分类（行业插件/知识库/问答库/解析算法/工具插件/核心插件/桌面端插件）
    - 所属行业（关联行业插件）
    - 标签（多个，用逗号分隔）
    - 图标（emoji 或 URL）
    - 兼容性要求（最低系统版本）
  - **状态设置**
    - 是否公开（公开/隐藏）
    - 是否推荐（精选插件）
    - 是否启用（上架/下架）
    - 是否废弃（标记已过时）
  - **评分与统计**
    - 评分（手动设置，1.0-5.0）
    - 下载量（手动校正）

- **版本管理**
  - **版本列表**
    - 查看插件的所有版本记录
    - 版本号、更新日志、文件大小、依赖信息
    - 是否为稳定版、是否启用
  - **创建版本**
    - 版本号（语义化，如 2.0.0）
    - 更新日志（Markdown 格式）
    - 下载 URL（插件包下载链接）
    - 文件大小（字节）
    - 依赖配置（JSON 格式，指定依赖的插件和版本约束）
    - 所需权限（逗号分隔的权限标识）
    - 是否稳定版标记
  - **版本管理操作**
    - 设为稳定版
    - 启用/禁用版本
    - 删除版本（保留版本记录）

- **安装统计**
  - **总览数据**
    - 总安装量
    - 本周新增安装
    - 本月新增安装
    - 安装趋势（折线图）
  - **安装历史**
    - 查看所有用户的安装记录
    - 按用户、插件、时间范围筛选
    - 查看安装详情（用户信息、安装时间、版本、状态）
  - **插件使用排名**
    - 按安装量排序的插件排行榜
    - 按行业分类的安装分布
    - 插件活跃度分析

- **批量操作**
  - 批量更新插件状态（上架/下架）
  - 批量设置推荐标记
  - 批量导出插件数据（CSV/Excel）

#### 数据字段
```
PluginPackage:
- id: 插件ID
- plugin_code: 插件代码（唯一标识）
- plugin_name: 插件名称
- description: 详细描述
- short_description: 简短描述
- category: 分类（industry/knowledge/qa/algorithm/tool/core/desktop）
- industry: 所属行业
- icon_url: 图标
- tags: 标签（逗号分隔）
- compatibility: 兼容性要求
- rating: 评分
- download_count: 下载量
- is_public: 是否公开
- is_featured: 是否推荐
- is_active: 是否启用
- is_deprecated: 是否废弃
- created_at: 创建时间
- updated_at: 更新时间

PluginVersion:
- id: 版本ID
- package_id: 关联插件ID
- version: 版本号
- changelog: 更新日志
- download_url: 下载链接
- file_size: 文件大小
- dependencies: 依赖配置（JSON）
- required_permissions: 所需权限
- is_stable: 是否稳定版
- is_active: 是否启用
- created_at: 创建时间

PluginInstallation:
- id: 安装记录ID
- user_id: 用户ID
- package_id: 插件ID
- version_id: 版本ID
- status: 状态（installed/uninstalled）
- config: 用户配置（JSON）
- installed_at: 安装时间
- uninstalled_at: 卸载时间
```
