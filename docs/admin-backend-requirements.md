# TonjClaw 营销网站 Admin 后台系统需求文档

## 1. 项目概述

### 1.1 项目背景
TonjClaw 营销网站 Admin 后台系统是与营销网站前端共享同一套服务架构的管理系统，用于管理员对系统进行全面管理和监控。

### 1.2 技术架构
| 层级 | 技术栈 |
|------|--------|
| 前端框架 | Vue 3 + Vite |
| UI组件库 | Element Plus |
| 状态管理 | Pinia |
| 路由管理 | Vue Router |
| HTTP客户端 | Axios |
| 图表库 | ECharts |
| 后端框架 | Flask (Python) |
| ORM | SQLAlchemy |
| 认证方式 | JWT Token |
| 数据库 | MySQL |

### 1.3 系统特点
- 与营销网站前端共享服务架构，无需前后端分离
- 支持角色权限控制（超级管理员、管理员、普通用户）
- 响应式布局，支持多端访问
- 模块化设计，易于扩展
