# TonjClaw Web 前端测试指南

## 🧪 快速测试

### 1. 启动服务

#### 后端服务（Flask）
```bash
# 在项目根目录
python main.py
# 或
start_web_server.bat
```

确保后端运行在 `http://localhost:5000`

#### 前端服务（Vite）
```bash
# 在 tonjclaw-web 目录
pnpm dev
# 或
start_web_frontend.bat
```

前端运行在 `http://localhost:5173`

## 🐛 常见问题排查

### 1. 登录失败
**症状**: 点击登录后无反应或报错

**检查**:
```javascript
// 打开浏览器控制台，查看是否有错误
// 检查 Network 面板，查看登录请求
```

**解决**:
- 确认后端服务已启动
- 检查 `.env.development` 中的 API 地址
- 确认数据库中有测试账号

### 2. 页面空白
**症状**: 访问页面显示空白

**检查**:
```javascript
// 打开控制台查看错误
// 检查路由配置是否正确
```

**解决**:
- 重启开发服务器
- 清除浏览器缓存
- 检查组件导入路径

### 3. 样式不生效
**症状**: Element Plus 组件样式异常

**检查**:
```javascript
// 确认 main.js 中导入了 Element Plus CSS
import 'element-plus/dist/index.css'
```

**解决**:
- 重新安装依赖：`pnpm install`
- 重启开发服务器

### 4. API 请求 404
**症状**: 所有 API 请求返回 404

**检查**:
- 后端是否运行在 5000 端口
- `.env.development` 配置是否正确

**解决**:
```env
# .env.development
VITE_API_BASE_URL=http://localhost:5000
```
